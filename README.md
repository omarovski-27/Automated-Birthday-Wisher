# Birthday Wisher

**Sends a beautiful Arabic + English birthday email to family and friends.**

---

## Features
- Reads `Birthdates.csv`
- Sends via Gmail (App Password)
- **Custom message inside code** (no files!)
- Runs automatically 
- Secure & clean

---

## Message (Arabic + English)

```text
Dear [Name],

عيد ميلاد سعيد يا [Name]! 
العمر كله يارب، وكل سنة وانت طيب!

May this year bring you health, happiness, and success.
Wishing you the best from Jordan!

With love,
Omar
